## 2. 退款申请流程

### 方式一：App / 官网（推荐）
1. 我的 → 我的订单 → 选择订单
2. 点击「申请退款」或「申请退货退款」
3. 选择原因，上传凭证（如有）
4. 提交后等待审核（1–2 个工作日）

### 方式二：天猫 / 京东
- 在对应平台订单页 → 申请退款/退货
- 按平台规则走，退款由原支付渠道退回

### 方式三：联系客服
- 热线 400-888-3366 或 service@starsleep.cn
- 提供：订单号 + 退款原因 + 照片/视频

## 4. 退款进度查询

- **App**：我的订单 → 退款/售后 Tab
- **状态**：申请中 → 审核通过 → 待退货（如需寄回）→ 退款中 → 已退款
- **天猫/京东**：平台订单 → 退款详情

---

## 5. 不支持退款的情况

1. 已拆封使用且非质量问题（超过 7 天无理由期）
2. 人为损坏（摔坏、进水、私自拆机）
3. 配件不全（缺充电线、保修卡等，可扣 ¥30 配件费后退主品）
4. 促销赠品未退回（需一并寄回或扣除赠品价值 ¥39）
5. 超过保修期且无质量问题的退货诉求

## 8. 常见问题

**Q：退款多久到账？**
A：微信/支付宝一般 1–3 个工作日到账；花呗约 1–7 个工作日恢复额度；银行卡 3–7 个工作日。若显示「退款成功」后超过 7 天仍未到账，请联系客服并提供支付截图。

**Q：退款被拒绝了怎么办？**
A：请先查看订单 → 退款详情中的拒绝原因（常见：超过 7 天无理由期、已使用无质量问题、配件不全）。如有异议，可回复订单号与情况说明申请人工复核，我们将在 3 个工作日内答复。

**Q：退款退到哪里？**
A：原路退回至付款账户，不支持换其他账户。

**Q：退货必须用顺丰吗？**
A：质量问题我们预约顺丰上门；无理由退货您可自选快递，保留单号即可。

**Q：只退一个，买了两个怎么办？**
A：订单详情中选择要退的商品和数量，支持部分退货。

**Q：优惠券会退吗？**
A：全额退款后，未过期优惠券退回账户；已过期不退。

**Q：换货算退款吗？**
A：不算。换货走「申请换货」通道，不经过退款流程。
