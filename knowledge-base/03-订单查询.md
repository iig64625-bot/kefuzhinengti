## 1. 订单状态说明

| 状态 | 含义 | 预计时间 |
|------|------|---------|
| 待付款 | 下单未支付 | 30 分钟未付自动取消 |
| 待发货 | 已付款，仓库备货中 | 付款后 24 小时内发货 |
| 已发货 | 快递已揽收 | 发货后 2–5 天送达（偏远 7 天） |
| 已签收 | 快递显示签收 | 签收后 7 天内可申请退换 |
| 已完成 | 交易完成 | 签收 7 天后自动完成 |
| 已取消 | 用户取消或超时未付 | — |
| 退款中 | 退款处理中 | 1–7 个工作日到账 |
| 已退款 | 退款成功 | — |

## 2. 自助查询订单

### 2.1 星眠 App 查询
1. 打开「星眠」App → 我的 → 我的订单
2. 切换 Tab：全部 / 待付款 / 待发货 / 待收货 / 已完成
3. 点击订单查看详情：商品、金额、物流、支付时间

### 2.2 天猫 / 京东查询
- **天猫**：淘宝 App → 我的淘宝 → 我的订单 → 搜索「星眠」
- **京东**：京东 App → 我的 → 全部订单 → 搜索「星眠」
- 第三方平台订单在对应平台查询，星眠 App 可能不同步

### 2.3 官网查询
1. 登录 www.starsleep.cn → 个人中心 → 我的订单
2. 输入下单手机号 + 验证码登录

## 3. 订单详情包含信息

- 订单号（格式：SS + 年月日 + 8 位数字，如 SS2026071312345678）
- 下单时间、支付时间
- 商品型号、数量、单价
- 优惠金额、实付金额
- 收货人、地址（部分隐藏）
- 支付方式（微信 / 支付宝 / 花呗）
- 物流公司和运单号
- 发票状态（如已申请）

## 4. 物流查询

### 4.1 查看物流
- App：我的订单 → 点击订单 → 查看物流
- 或复制运单号到快递公司官网/微信物流助手查询

### 4.2 合作快递
- 默认：**顺丰速运**（满 ¥500 包邮）
- 偏远地区：中通 / 邮政

### 4.3 物流常见问题

**Q：付款很久了还没发货？**
- 大促期间可能延迟至 48 小时，请耐心等待
- 超 48 小时未发：联系客服加急，或申请退款

**Q：物流一直不更新？**
- 揽收后 24 小时无更新：联系客服核实
- 可能滞留中转站，客服可帮您催促快递

**Q：显示签收但没收到？**
1. 先问家人/同事/物业是否代收
2. 查看快递柜、驿站取件码
3. 仍未找到：联系客服，我们协助联系快递核实

**Q：能改收货地址吗？**
- **待发货**：App 订单详情 → 修改地址（或联系客服）
- **已发货**：无法修改，可联系快递转寄（运费自理）或拒收后重拍

**Q：能指定快递吗？**
- 暂不支持指定，默认顺丰。如有特殊需求请联系客服备注（不保证满足）。

## 5. 支付与发票

### 5.1 支付方式
- 微信支付、支付宝、花呗（天猫/京东以平台为准）
- 不支持货到付款

### 5.2 支付异常

| 问题 | 处理 |
|------|------|
| 扣款了但订单待付款 | 等 5 分钟刷新；仍异常请联系客服+支付截图 |
| 重复扣款 | 提供截图，核实后 1–3 工作日退多扣款项 |
| 想取消待付款订单 | 订单详情 → 取消订单 |

### 5.3 发票
- **电子发票**：订单完成后，App → 订单详情 → 申请发票
- 类型：增值税普通发票（电子）
- 内容：明细（商品名称）或大类（日用品）
- 时效：申请后 3 个工作日内发送至邮箱
- 已开票订单：订单详情 → 查看发票 / 重新发送

## 7. 常见订单 FAQ

**Q：订单号在哪里看？**
A：App/官网 → 我的订单 → 点进订单，顶部即订单号。天猫/京东在平台订单详情中查看。

**Q：帮别人买的能用我的账号查吗？**
A：订单绑定下单账号。代查需下单人授权或提供订单号+收货手机号核实。

**Q：预售订单什么时候发？**
A：商品页和订单页会标注预计发货日期，一般付尾款后 3 天内发出。

**Q：能合并两个订单发货吗？**
A：待发货状态联系客服尝试合并，已打包则无法合并。

**Q：订单删除了还能查吗？**
A：App 删除仅隐藏显示，联系客服可恢复查询。平台订单在淘宝/京东仍可查。
