# 可观测性指南

## 1. 对话日志模板

每次测试或演示后记录一行，可复制到 Excel / `observability-log.csv`：

| 时间 | 用户问题 | 意图类型 | 检索命中 | 引用文档 | 回答质量 | 是否转人工 | 备注 |
|------|---------|---------|---------|---------|---------|-----------|------|
| | | | Y/N | | 优/良/差 | Y/N | |

## 2. 关键指标

| 指标 | 计算方式 |
|------|---------|
| 检索命中率 | 引用文档非空次数 / 总轮次 |
| 空检索率 | result=[] 次数 / 总轮次 |
| 转人工率 | 转人工次数 / 总轮次 |
| 用户满意度 | 👍 / (👍+👎)（若开启反馈） |

## 3. Dify 后台

- **日志与标注**：查看历史对话
- **标注**：对错误回答打标，用于迭代
- 截图 2～3 条脱敏对话放入 `docs/screenshots/`

## 4. 失败样本库

见 [failure-cases.md](failure-cases.md)

## 5. 告警阈值（作品集可写「规划」）

| 指标 | 阈值 | 动作 |
|------|------|------|
| 空检索率 | >20% | 检查 KB 索引 |
| 转人工率 | >40% | 补充 FAQ |
| 平均响应 | >15s | 检查模型/TopK |
